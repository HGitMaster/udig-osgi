1 Code Examples

We have gathered up the code examples used in the developers guide into a single plug-in:

  • net.refractions.udig.tutorials.examples

Reading the Code Examples

You can view the code examples directly from svn:

  • 1.1.x examples

Import Examples from the SDK

You can import the examples as they were when your SDK was produced:

  1. File > Import from the menu bar
  2. Plug-in Development > Plug-ins and Fragments as the input source
  3. On the Import Plug-ins and Fragments page
    • The sdk directory will be listed as your target platform
    • Import As: Porjects with source folders
  4. On the Selection page
  5. select net.refractions.udig.tutorials.examples from the list and press the Add button
  6. Press Finish

For the UDIG 1.1-RC14 release you will need to import the projects as binary; the src folder will still be there - it was just packaged up strangely.

Checking out Examples from SVN

You can use svn to check out the examples into workspace:

  1. Use the command line svn tool: 
    C:\java\sdk>
svn co http://svn.geotools.org/udig/branches/1.1.x/udig/tutorials/net.refractions.udig.tutorials.examples examples
  2. Select File > Import
  3. Choose General > Existing Projects into Workspace
  4. From the Import Projects page browse to: C:\java\sdk\examples
  5. Choose net.refractions.udig.code.examples from the list
  6. Press Finish
