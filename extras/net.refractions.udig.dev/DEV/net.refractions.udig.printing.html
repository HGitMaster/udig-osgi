net.refractions.udig.printing

The printing engine takes a net.refractions.udig.printing.page (that is a net.refractions.udig.printing.context and net.refractions.udig.render.decorator ) and aPrinter and begins a specific rendering pipeline.

Functional Requirements
  • Maintains a cache
  • Process Page + Printer
  • Constructs what is to be printed from a net.refractions.udig.printing.context with additional Metadata and Connection info from Data Manager
  • Passes each layer off to its appropriate Renderer along with the Graphics2D from the Printer
  • Print Progress/Cancel
Non-functional Requirements
Design Notes
  • Pringing Engine is "smart" - determines whether a given WMS and WFS share the same source and can use either to retrieve the same data. For printing, this will allow it use WFS when available, as that will be much faster in response time.
  • May need to separate out WMS requests into separate calls tiled together on the client side at native printer resolution
