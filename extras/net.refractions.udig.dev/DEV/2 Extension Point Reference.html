SDK Search

The same information is included as part of your SDK, to locate an extension point definition:

  1. Select Search > Search... from the menu bar
  2. Change to the Plug-in Search Tab
    • Search string: tool
    • Search for: Extention Point
    • Limit To: Declarations
    • External Scope: Enabled Plugin-ins only
    • Scope: Workspace
  3. Press the Search button
  4. The Search view is opened showing the results of your query
  5. Right click on net.refractions.udig.project.ui.tool and select Show Description
