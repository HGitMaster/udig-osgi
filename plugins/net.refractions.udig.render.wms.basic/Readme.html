Basic WMS Renderer Implementation

Purpose:

This plugin does not have a public API. It is the default WMS Renderer plugin.

A WMS renderer handles the communication with a WMS and normally uses a GridCoverageRenderer to display the data. A WMS renderer is seperate from the GridCoverageRenderer because multiple layers maybe rendered by a WMS with one request. The WMSRenderer is a renderer plugin that has (or uses) a WMS client. The WMS Renderer must know how to create the necessary parameter from the Layers it is intended to create.
