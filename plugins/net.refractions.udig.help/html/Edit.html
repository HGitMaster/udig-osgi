Edit

In order to edit a feature, you must have a map open and selected.

  1. Create a map using the sample lakes information.
  2. Zoom into a small area of the map.
  3. Select the layer in the Layers view that you wish to edit
  4. Using the Select Features by Drawing a Rectangle tool, select a lake. (The selected feature will flash)
  5. Change to the Feature Edit tool.
  6. You can use the Feature Edit tool to move points. Move your mouse over a vertex. The cursor will change to four arrows. Click and drag the vertex. Once you release, the feature will be redrawn.
  7. You can also create new points with the Feature Edit tool. Click on a straight line where there is no vertex. A vertex will appear and it can be moved.

You may also wish to look at the Feature Editor, in order to change attribute information. You can use Delete Feature tool to remove features. Creating new features is done by selecting the Feature Edit tool when there are no features selected and clicking on the map.
