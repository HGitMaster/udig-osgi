Adding Features from ArcSDE

To import data from ArcSDE you can either :

  • Press the first button on the toolbar to show the Add Layer Wizard; Or
  • Select the Add item from the Layer menu to show the Add Layer Wizard. Or
  • Select the File > New > Layer from the file menu

These actions open up the Add Layer wizard.

  1. Select ArcSDE from the Wizard
  2. Enter your ArcSDE connection parameters
    Name Description
    host Server name (or ip address) where the ArcSDE gateway is running
    port Port number in wich the ArcSDE server is listening for connections.Generally it's 5151
    username The name of a valid database user account
    password The database user's password
    database the specific database to connect to. Only applicable to certain databases. Value ignored if not applicable.
  3. Press Next
  4. Select layer(s) that will compose the new map
  5. Press Finish

ArcSDE features will be added to the Catalog and a new map containing selected Layers will show up.
