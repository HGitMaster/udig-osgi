Creating a new Map

There are numerous ways to create a new map:

  • Select the New->New Map item under the File Menu and a new blank map is created.
  • Press the arrow beside the new button in the Main toolbar and select New Map . A new blank map will be created
  • Drag a file or URL from the file system or a browser onto an empty Map Editor or empty Layers View. A new map containing layers made up from the dragged resource will be created.
  • Right click on a service or resource in the Catalog View or the Search View and select Add to New Map . A new map containing layers made up from the dragged resource will be created.
  • Drag a service or resource from the Catalog View or the Search View onto an empty Map Editor or empty Layers View. A new map containing layers made up from the dragged items will be created.
  • If there are no maps open select New Layer from the new button in the toolbar or from the New menu under the File Menu or select Add Layers from the Layers Menu item. Completing the wizards will created a new map.
