Adding Features from a DB2

To import data from a DB2 you can either :

  • Press the first button on the toolbar to show the Add Layer Wizard; Or
  • Select the Add item from the Layer menu to show the Add Layer Wizard. Or
  • Select the File > New > Layer from the file menu

These actions open up the Add Layer wizard.

  1. If you have not set your DB2 preference (Window > Preferences...) then you will have the chance to do so now.
    See DB2 Preferences for instructions on how to do this.
  2. Select DB2 from the Wizard
    Enter your DB2 host and authentification parameters
    Select the database connect and optionaly a schema
    Press Next
  3. Select layer(s) that will compose the new map
    Press Finish

DB2 Layers will be added to the Catalog and a new map containing selected Layers will show up.
