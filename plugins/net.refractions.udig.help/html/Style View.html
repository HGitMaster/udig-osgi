Style View


The style view allows layers to be styled differently. The toolbar contains a drop-down list that list all the types of styles available, an example being an SLD style. The toolbar also provides access to editors for the different aspects of a particular style. For example, an SLD style can have polygon symoblizers and Text symbolizers and separate editors exist for each. Each is a drop down button with a list of the different editors for that aspect and also permits that aspect of the style to be disabled.
