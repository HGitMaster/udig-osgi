Using the New button

We will now create a new map for our project using the toolbar New button.

  1. On the toolbar, click the New button.
  2. A dialog will open allowing you to choose the type of data you want to add to a map.
  3. Select Files and press Next
    (or double click on Files)
  4. Browse to your sample data directory (sample data)
  5. Open the directory and select the file lakes.shp
    (You can select more than one file at a time)
  6. Press Open

A new map (called Map1) will be opened, contained in the last selected Project.
