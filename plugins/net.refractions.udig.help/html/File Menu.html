File Menu

New

New Menu Action
New Project Create a new project
New Map Create a new Map
New Layer Create a new Layer

Open Project

Open an existing project

Close Project

Close a project. This menu is only available when a project is already opened.

Close

Close the current map. In case map has unsaved changes user will be asked to save change or not. This menu is only available when a map is already opened.

Close All

Close all maps. In case map has unsaved changes user will be asked to save change or not. This menu is only available when a map is already opened.

Save

Save current map. This menu is only available when a map is already opened.

Save As

Save project with user specified filename and path.

Save All

Save all opened maps

Commit

Commit changes to server/database

Roolback

Commit changes from server/database

Refresh

Refresh the map editor

Print

Print map

Rename

Rename Project or layers that are selected from the project view

Properties

Get project properties. This will allow user to set the project Coordinate Reference System from the project preference page.

Exit

Quit the uDig application
