Layer wizard

The layer wizard can be used to add layers to the currently open Map (a map will be created if non exists).

  1. Create a new new map using File > New > Map
  2. Bring up the Layer wizard by selecting Layer > Add... or File > New > Layer
  3. Choose the kind of data type you wish to add from the avaiable list.
    We will choose a Web Map Server for this example, select Web Map Server and press Next.

    You may also proceed by double clicking on Web Map Server.
  4. Enter in the following GetCapabilities url:
    http://wms.jpl.nasa.gov/wms.cgi?Service=WMS&Version=1.1.1&Request=GetCapabilities

    And choose Next
  5. Select "modis" from the provided list
  6. Click Finish when done, a new map is created with the selected layer.
