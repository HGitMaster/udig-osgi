Catalog view

The Catalog is a uDig view that contains known data that can be added to a map. It can include many types of information, including Shapefiles, Web Map Servers, and Grid Coverage Exchanges.

Right clicking on an object allows you to perform actions on it, such as adding it to a new or existing map. To see a description of what each menu item does, move selection highlight to that menu item and press the context-sensitive help key (e.g., F1 on Microsoft Windows.)

Toolbar

The toolbar of the Catalog view contains the following buttons:
Import - Brings up the Import Wizard to add data to the Local Registry.
Remove - Removes the selected object from the Local Registry.
