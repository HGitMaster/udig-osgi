Adding layers from a URL

To import layer(s) from a URL (WFS/WMS/local file) you can either :

  • Press the first button on the toolbar to show the Add Layer Wizard; Or
  • Select the Add item from the Layer menu to show the Add Layer Wizard. Or
  • Select the File > New > Layer from the file menu

These actions open up the Add Layer wizard.

  1. Select URL from the Wizard and press Next
  2. Enter your WFS/WMS/local file URL or select recent one and press Open
    WMS links can be found here or Test servers
    WFS link can be found here
  3. Select layer(s) that will compose the new map and press Finish
  4. WMS Layers will be added to the Catalog and a new map containing selected layers will show up.
