Using the Context menu

After creating a map, we can add some data to it using the context menu.

1. Right click on the recently created map

2. Select "Add...".
3. Select "Files" from the list of available wizards.
4. Browse to your sample data directory (sample data)
5. Open the directory and select one or more of the files available.
6. Press "Open"

The shapefiles will be added to the map and will be displayed on the screen.
