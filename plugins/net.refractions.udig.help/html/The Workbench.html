The workbench

When the application is launched, a single Workbench window is displayed.

When you start up uDig for the first time the Welcome view is displayed. Click the arrow labeled Workbench, to reveal the uDig workbench. You can return to the Welcome view at any time by selecting the Help menu and choosing Welcome.

A Workbench window offers multiple editors and views, such as the Projects view. Multiple Workbench windows can be opened simultaneously.

In the above example a single Map is open along with the Projects, Layers and Search views.
