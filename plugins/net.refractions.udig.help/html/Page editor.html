Page editor
The Page Editor allows manipulation of a page to be used for printing.

Boxes can be moved around and resized within the page. Certain boxes can be edited by single clicking on them or right clicking and bringing up the context menu.

Eventually, other boxes can be created through the use of the palette.
The page can be printed by pressing the print button.
