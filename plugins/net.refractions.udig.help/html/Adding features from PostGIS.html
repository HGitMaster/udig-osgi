Adding Features from PostGIS

To import data from PostGIS you can either :

  • Press the first button on the toolbar to show the Add Layer Wizard; Or
  • Select the Add item from the Layer menu to show the Add Layer Wizard. Or
  • Select the File > New > Layer from the file menu

These actions open up the Add Layer wizard.

  1. Select PostGIS from the Wizard
    Enter your PostGIS host and authentification parameters
    Select the database connect and optionaly a schema
    Press Next
  2. Select layer(s) that will compose the new map
    Press Finish

    PostGIS Layers will be added to the Catalog and a new map containing selected Layers will show up.
