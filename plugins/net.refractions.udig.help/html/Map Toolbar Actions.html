Button Command Description
print creates a page for printing based on the current map
     
  Dynamic Help  
  Search Help  
     
  back Navigation command returning to previous map location
  forward Navigation command restoring a map location changed using the back command
     
refresh map redraws the current map
cancel cancel the current rendering
show all A navigation command, show the full map
zoom in A navigation command, decrease the Map scale by a fixed amount
zoom out A navigation command, increase the Map scale by a fixed amount
Tool Category Description
zoom A navigation tool (shortcut 'z') providing control of Map scale.
pan A navigation tool (shortcut 'p') providing control of Map location.
info Request information (shortcut 'i') for content at an indicated location.
edit An edit tool (shortcut 'e') allowing selection and modification of polygon, point or line geometries
add vertex An edit tool (shortcut 'e') allowing the addition of vertices to polygon or line geometries.
remove vertex An edit tool (shortcut 'e') allowing the removal of vertices from polygon or line geometries.
hole An edit tool (shortcut 'e') allowing the creation and modification of holes within polygons.
polygon creation An create tool (shortcut 'c') for creating polygons.
line creation An create tool (shortcut 'c') for creating lines.
point creation An create tool (shortcut 'c') for creating points.
draw geometry An create tool (shortcut 'c') for drawing polygon or line geometries.
ellipse An create tool (shortcut 'c') for creating line or polygon ellipses.
rectangle An create tool (shortcut 'c') creating line or polygon rectangles.
fill An feature edit tool (shortcut 'f') for filling between other geometries.
split An feature edit tool (shortcut 'f') for splitting existing geometries.
delete An feature edit tool (shortcut 'f') allowing the removal of Features.
Button Command Description
commit outgoing sync
revert rollback and clear current changes
     

Navigation commands and Tools retain a location history that one can navigate through using the Prevision and Next buttons.
