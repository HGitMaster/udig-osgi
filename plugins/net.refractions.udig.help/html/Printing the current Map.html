Printing the current Map

To print the current map you must first create a Page that allows you to lay it out and print it from there.

  1. Press the print button on the toolbar to create a page.
  2. It will ask you what template you wish to create the page from. At this time there is only one template to choose from. Select one and press OK.
  3. From there you can press the print button again to send the page to the printer.
