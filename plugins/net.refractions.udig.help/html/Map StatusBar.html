Map StatusBar

Status Bar Item Description Example
Coordinate Reference System Display Displays the Current Projection of the Map. Clicking on the button will open a dialog allowing the projection to be changed
Mouse Position Displays the current or last location of the Mouse in the Map Editor. Entering 2 numbers seperated by spaces or commas will center the map editor at that location. If LL or LATLONG (case doesn't matter) is at the end, the position will be assumed to be in WGS84 and will be correctly transformed
Progress Indicator Indicates that background jobs are running. The label is the title of the job, the animated image shows that the job is running and clicking on the button will open a view that shows the current state of all running jobs in the system
