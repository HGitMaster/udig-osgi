Working with Data

To work with spatial data (layers or features) in the uDig application you may either:

  • Press the first button on the toolbar to show the Add Layer Wizard; Or
  • Select the Add item from the Layer menu to show the Add Layer Wizard. Or
  • Select the File > New > Layer from the file menu

These actions open up the Add Layer wizard.

To import spatial data into the udig evnironment for later use you may either:

You can use the Import Wizard to copy data into the applications catalog.

  1. From the main menubar, select File > Import
  2. Select Data and click Next
  3. Choose from the available data formats and click Next
  4. Your imported data is now available in the Catalog view

Tips: You can also import files and urls by dragging them from the file system, or web browser, and dropping them in the Catalog view, or by copying and pasting.
