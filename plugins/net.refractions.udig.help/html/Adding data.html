Adding Data

Data can be used with uDig either by:

  • dragging and dropping from the file system, or web browser
  • dragging and dropping from a udig view
  • Adding layers to a map Layer > Add... (or File > New > New Layer )

Using drag and drop or copy/paste relies on the operating system and it not necessarilly available on all platforms. If the platform you are using does not have this support, you can always use the Layer > Add... wizard.
