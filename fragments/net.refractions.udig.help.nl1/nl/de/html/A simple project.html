Ein einfaches Projekt

Nachdem die wichtigsten Elemente des Arbeitsplatzes erklärt wurden, erhalten Sie hier eine Anleitung zum Erstellen eines einfachen Projekts. Neue Projekte und Karten kann man, wie hier gezeigt wird, auf vielerlei Art erstellen:

  • Mittels Dateimenü
  • Mittels der Schaltfläche "Neu"
  • Mittels des Kontextmenüs

Mittels Dateimenü

Sie können ein neues Projekt über den Menüeintrag Datei > Neu im Menü des Arbeitsplatzes anlegen. Folgen Sie für ein einfaches Projekt folgenden Schritten:

  1. Wählen Sie Datei > Neu > Projekt
  2. Geben Sie im Feld Neues Projekt einen Dateinamen ein. Sie können natürlich auch die Schaltfläche Durchsuchen verwenden.
  3. Klicken Sie dann auf Fertigstellen.

Im Projektfenster ist nun Ihr neues Projekt eingetragen.

Mittels der Schaltfläche "Neu"

Wir werden nun eine neue Karte für das Projekt erstellen und diesmal den
Symbolleisteneintrag Neu verwenden.

  1. In der Symbolleiste des Arbeitsplatzes befindet sich die Schaltfläche Neu .
  2. Es öffnet sich nun ein Dialog zum Auswählen des Datentypes, der zur Karte hinzugefügt werden soll.
  3. Wählen Sie Dateien und klicken Sie auf Weiter. (Ein Doppelklick auf Dateien tut's auch.)
  4. Durchsuchen Sie Ihre uDig-Installationsverzeichnis (i.d.R. "C:\Program Files\uDig\eclipse\")
  5. Öffnen Sie das Verzeichnis "shapefiles" und wählen Sie eine oder mehrere der verfügbaren Dateien.

Im aktuellen Projekt wird nun mit den gewählten Daten eine neue Karte erzeugt. Diese hat den Namen "Map1", kann aber umbenannt werden.

Mittels Kontextmenü

Nachdem eine Karte erstellt wurde, werden wir ihr mittels Kontextmenü weitere Daten hinzufügen.

  1. Klicken Sie mit der rechten Maustaste auf den gerade erstellten Karteneintrag im Projektfenster.
  2. Wählen Sie Hinzufügen ....
  3. Wählen Sie "Dateien" aus der Liste der verfügbaren Assistenten.
  4. Durchsuchen Sie Ihre uDig-Installationsverzeichnis (i.d.R. "C:\Program Files\uDig\eclipse\")
  5. Öffnen Sie das Verzeichnis "shapefiles" und wählen Sie eine oder mehrere der verfügbaren Dateien.
  6. Bestätigen Sie die Auswahl mit "Öffnen".

Die Shapefiles werden nun der Karte hinzugefügt und auf dem Bildschirm dargestellt.

Natürlich existieren weitere Möglichkeiten Ressourcen wie Projekte, Karten und Daten hinzuzufügen. Die oben genannten sind lediglich Beispiele.
