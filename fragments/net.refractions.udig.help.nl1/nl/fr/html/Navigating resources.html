Naviguer les ressources

Naviguer dans la vue Projets

La vue Projets permet aux utilisateurs de naviguer les projets et les cartes dans le Workbench. Chaque projet peut comprendre de multiples cartes et des autres ressources. Elles sont les éléments principaux de la vue Projet.

Pour ouvrir une carte ou un editeur de page depuis la vue projets:

  • Double cliquer sur une carte ou une page
  • Selectionner la carte (ou Page), effectuer un clic droit pour afficher le menu contextuel et selectionner Ouvrir

Naviguer dans la vue Catalogue

La vue Catalogue permet aux utilisateurs de visualiser toutes les données importés à uDig. À l'aide de cette vue, les utilisateurs peuvent inspecter les données davantage et les ajouter à la carte en cours ou à une nouvelle carte.

Pour essayer d'autres vues, consulter la liste en sélectionnant le menu Window > Show View.

La vue Catalogue présente l'information divisée en:

  • Service: par exemple Serveurs d'Entités Web, bases de données, nom de fichiers
  • Couches: Les couches incluent le contenu d'un fichier Shape, une couhce individuelle d'un Serveur Web Cartographique

Il existe plusieurs possibilités pour utiliser des couches via la vue Catalogue:

  1. Selectionner une couche et effectuer un clic droit pour afficher le menu contextuel:
    • Choisir Ajouter à une nouvelle carte pour créer un carte basée sur cette couche
    • Choose Ajouter à la carte courante pour utiliser ce contenu avec votre éditeur courant
  2. Glisser et déposer une couche dans l'éditeur courant ou la vue Projets.

Naviguer dans la vue Recherche

La vue catalogue est utilisée pour explorer les informations des couches ayant déjà été localisées. La vue Recherche est utilisée pour rechercher des informations disponibles sur des catalogues publics comme le portail Geo Connections.

  1. Ouvrir la vue Recherche
    (si celle ci n'est pas déjà à l'écran, vous pouvez la trouver dans Window > Views > Search)
  2. Entrer un critère de recherche : dans cet exemple, nous utiliserons le mot birds
  3. Selectionner n'impote quel couche. Vous pouvez effectuer un clic droit pour affiche rle menu contextuel ou faire un glisser déposer pour ajouter cette information à la carte
