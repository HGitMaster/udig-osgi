Créer une nouvelle carte

Il existe différents moyens de créer une carte :

  • Selectionner Nouveau->Nouvelle Carte depuis le menu fichier. Une nouvelle carte vide est crée.
  • Selectionner la flèche à coté du bouton Nouveau de la barre de d'outils et sélectionner Nouvelle Carte. Une nouvelle carte vide est crée.
  • Faites un glisser-Déposer d'un fichier, d'une URL du système de fichier vers un Map Editor ou Layers View vide. Une nouvelle carte contenant les couches ajoutées par la fonction glisser-déposer est crée.
  • Faites un clic droit sur un service ou une ressource dans la vue Catalogue Catalog View ou la vue recherche Search View et sélectionner "Ajouter à une nouvelle carte". Une nouvelle carte contenant les couches ajoutées est crée.
  • Faites un glisser-Déposer d'un service ou d'une ressource de la vue catalogue Catalog View ou la vue recherche Search View vers un Map Editor ou Layers View vide.Une nouvelle carte contenant les couches ajoutées par la fonction glisser-déposer est crée.
  • S'il n'y a aucune carte ouverte, selectionner Nouvelle couche depuis le boutton Nouveau dans la barre d'outils ou depuis l'élément Nouveau du menu fichier ou selectionner Ajouter couche depuis le menu Couches.
