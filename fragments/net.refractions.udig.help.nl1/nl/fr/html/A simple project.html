Un projet simple

Les composants de base du Workbench expliqués, les utilisateurs peuvent se mettre à créer un simple projet. Il existe plusieurs façons de créer des nouveaux projets et des nouvelles cartes. Dans cette section du tutoriel, les ressources seront manipulées les trois manières suivantes :

  • Le menu Fichier
  • Le bouton Nouveau
  • Le menu Contexte

Les utilisateurs peuvent créer un projet en utilisant le menu Fichier. Une fois le projet est créé, une nouvelle carte peut aussi être créée, et des données y ajoutées par la suite. Il existe beaucoup plus de façons à créer des ressources; les trois ci-haut ne sont que des exemples.
