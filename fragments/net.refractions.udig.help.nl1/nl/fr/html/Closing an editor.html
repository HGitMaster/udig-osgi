Fermer un éditeur

Maintenant que nous avons un éditeur ouvert, voici comment le fermer.

  1. Pour fermer un éditeur ouvert, choisisser une des options suivantes:
    • Cliquer le bouton « X »: .
    • Selectionner Fichier > Fermer depuis la barre de menu
  2. Remarquer qu'uDig vous demande si vous souhaitez enregistrer la carte avant de fermer l'éditeur
  3. Cliquer OK pour sauvegarder les changements et fermer l'éditeur.
