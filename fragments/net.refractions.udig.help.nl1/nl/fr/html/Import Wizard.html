Cette option permet l'importation d'un fichier Shape.

L'importation - Les options pour les fichiers Shape
L'option La description L'option par défaut
L'URL L'URL d'un fichier Shape
tampon memorisé actif rend désactif les entrées-sorties projetées en mémoires Vrai pour l'URL d'un fichier
Faux pour l'URL de protocole HTTP

Les URLs de fichiers utilisent par défaut un tampon memorisé, mais cette option n'est pas offerte pour les URLs de protocole HTTP. Une importation effectuée à l'aide d'un tampon memorisé sera réalisée beaucoup plus vite, mais pour importer des fichiers très larges (200 megs) les utilisateurs voudraient peut-être forcer à "faux" cette valeur.

L'importation - Les options pour les serveurs d'entités Web
Les options L'abileté L'option par défaut
GET_CAPABILITIES_URL L'URL d'un document GetCapabilities d'un serveur d'entités Web
(C'est l'URL à utiliser dans le navigateur Web.)
SERVER_URL L'URL d'un service d'entités Web
(L'URL incomplet pour lequel on doit préparer un « capability ».)
Les options La méthode de requête L'option par défaut
USE_POST Utiliser "Post" pour la requête de contenu; les requêtes complètes sont permises
USE_GET Utiliser "Get" pour la requête de contenu; peut être enregistrée
Les options L'authentification L'option par défaut
NOM D'UTILISATEUR Nom d'utilisateur pour l'authentification de protocole HTTP
MOT DE PASSE Mot de passe pour l'authentification de protocole HTTP

La plus simple méthode à suivre :

1. À l'aide d'un navigateur Web, trouver le document GetCapabilities.
2. Copier et coller l'adresse Web au GET_CAPABILITIES_URL.
3. Si nécessaire, entrer le nom d'utilisatuer et le mot de passe.
4. Cliquer « Terminer ».
