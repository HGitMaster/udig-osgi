Les outils

Les outils, tels que ceux dans la liste ci-dessous, facilitent la manipulation d'une carte:

  • Le zoom
  • La fonction panoramique
  • L'édition

En fonction de l'outil sélectionné, la carte va se comporter de différentes manières.

L'outil en cours est indiqué par un fond grisé. Les outils sont regroupés en accord avec les catégories suivantes: (hotkey in brackets):

  • ( z ) Zoom: change l'echelle de la carte
  • ( p ) Pan: change la localisation de la carte
  • ( s ) Selection: permet à l'utilisateur de sélectionner l'information voulue
  • ( i ) Info: permet l'identification du contenu de la carte
  • ( e ) Edit: permet la manipulation et la création d'informations

Des outils additionnels sont disponibles dans chaque catégories. Cliquer sur le petit triangle pour afficher les outils additionnels:

Les raccourcis clavier peuvent être utilisés pour changer de catégorie d'outil et pour changer d'outil dans la catégorie courante..
