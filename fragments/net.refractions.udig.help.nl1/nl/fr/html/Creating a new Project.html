Créer un nouveau Projet

Pour créer un nouveau Project:

  1. Selectionner Nouveau->Nouveau Projet depuis le menu fichier
  2. Entrer un chemin comme C:\uDig Projects\test Project.
    • ".udig" est ajouté à la fin du chemin s'il n'est pas déjà présent.
    • Vous pouvez utiliser le boutton Naviguer pour localiser un répertroire. Entrer le nom du fichier et sélectionner Ouvrir.

Le projet a été crée. Il maintenant disponible dans l'explorateur de projets
