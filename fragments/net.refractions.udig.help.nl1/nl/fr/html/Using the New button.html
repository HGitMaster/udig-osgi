Utiliser le bouton Nouveau

Nous allons maintenant créer une nouvelle carte pour le projet en utilisant le bouton Nouveau dans la barre d'outils :

  1. Dans la barre d'outils, cliquer sur le bouton Nouvelle carte .
  2. Une boite de dialogue s'ouvre vous permettant de choisir le type de données que vous souhaiter ajouter à la carte.
  3. Selectionner Fichiers et appuyer sur Suivant
    (ou double cliquer sur Fichiers)
  4. Naviguer jusqu'au repertoire d'installation d'uDig
    (typiquement C:\Program Files\uDig)
  5. Ouvrir le repertoire des fichiers Shape et selectionner le fichier lakes.shp
    (Vous pouvez selectionner plusieurs fichiers à la fois)
  6. Appuyer sur Ouvrir

Une nouvelle carte vide s'ouvrira dans le projet sélectionné en dernier.
