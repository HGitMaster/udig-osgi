Accéder à l'aide contextuelle

Pour accéder à l'aide contextuelle pour un objet d'une fenêtre, selectionner l'objet en lui donnant le focus (voir table ci dessous) puis appuyer sur la touche F1. Pour les systèmes utilisant GTK, utiliser Ctrl+F1.
Cela affiche une fenêtre popup d'information avec la description de l'objet sélectionné et, habituellement, une liste de liens vers d'autres informations en rapport. Vous pouvez fermer la fenêtre popup en cliquant en dehors ou en appuyant sur Echap.

Si vous voulez plus d'informations que ce qui est décrit dans la fenêtre popup, cliquer sur un lien vers d'autres informations en rapport. Cette action ouvre le navigateur d'aide sur le sujet sélectioné et ferme la fenêtre popup. La liste des liens reste dans la frame de gauche (pas besoin d'appuyer à nouveau sur F1 si vous êtes interessé par d'aures sujets en rapport avec le même objet.

La table suivante montre comment donner le focus à différents types d'objets.

Type d'objet Comment le sélectionner
Element de menus (ex. Sauvegarder du le menu fichier) Laisse le pointer de la souris sur l'objet pour qu'il soit selectionné mais sans cliquer dessus.
Champ Mettre le curseur dans le champ.
Boutton (ex. Boutton Annuler d'une boite de dialogue) Utilise la touche Tab jusqu'à ce que le bouton ait le focus.
Liste Cliquer sur un élément de la liste.
Vue ou Panneau Cliquer sur la barre de titre de la vue ou du panneau.
Case à cocher Cliquer sur la case à cocher (qui change d'état) ou utiliser la touche Tab jusqu'à lui donner le focus.

Note: l'aide contextuelle via F1 n'est pas disponible depuis les boutons de la barre d'outils. Au lieu de cela, laisser votre pointeur de souris planer au dessus d'un bouton de la barre d'outils pour voir l'aide du boutton.
