Les vues rapides

Les vues rapides sont cachées par défaut, mais les utilisateurs peuvent rapidement les rendre visibles. Elles fonctionnent tout comme les vues regulières, sauf qu'elles sont cachées pour limiter l'espace qu'elles occupent dans la fenêtre Workbench.

Cette section explique comment convertir la vue Projets en vue rapide.
