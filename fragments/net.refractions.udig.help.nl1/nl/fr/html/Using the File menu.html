Utiliser le menu Fichier

Les utilisateurs peuvent créer un projet en cliquant sur le menu Fichier > Nouveau dans la barre de menus du Workbench. Créer un simple projet en suivant ces étapes :

  1. Dans la barre de menus, cliquer sur Fichier > Nouveau > Projet
  1. Dans le champ Nouveau projet , entrer un nom de fichier.
    (Les utilisateurs peuvent aussi se servir du bouton Naviguer .)
  2. Cliquer Terminer une fois terminé.

Remarquer que la vue Projets contient maintenant le nouveau projet.
