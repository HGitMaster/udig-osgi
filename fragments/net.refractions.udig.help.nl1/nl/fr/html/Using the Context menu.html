Utiliser le menu Contextuel

Après avoir créer une carte, les utilisateurs peuvent y ajouter des données en se servant du menu Contextuel.

1. Faire un clic droit sur la carte récemment créée.

2. Sélectionner « Ajouter... ».
3. Sélectionner « Fichiers » de la liste d'assistants offerts.
4. Naviguer jusqu'au répertoire d'installation de uDig (en général, C:\Program Files\uDig\eclipse).
5. Ouvrir le répertoire de fichiers Shape et sélectionner un ou plusieurs des fichiers disponibles.
6. Cliquer sur "Ouvrir".

Les fichiers Shape seront ajoutés à la carte et affichés à l'écran.
